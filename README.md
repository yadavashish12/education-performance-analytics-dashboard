# Education Performance Analytics Dashboard

This project is a complete end-to-end Power BI dashboard developed using a business wireframe and KPI requirements document.

---

## Dashboard Focus

- Education program monitoring
- KPI tracking
- Performance analytics
- Stakeholder reporting
- Interactive drill-down insights

---

## Tech Stack

- Power BI
- SQL
- DAX
- Power Query
- Excel

---

## Key Features

- Interactive dashboards
- KPI cards and trend analysis
- Automated data transformations
- Data modeling and relationships
- Business-focused storytelling
- Multi-page reporting structure

---

## Business Use Case

The dashboard simulates a real-world analytics environment used in education governance and large-scale program monitoring.
